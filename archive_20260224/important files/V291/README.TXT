The following directorys contain the demos for the DoPE interface:

DEMO.VB             Visual Basic
DoPENet_CSharpDemo  .NET C# Demo
DoPENet_VBDemo      .NET Visual Basic Demo
DEMO.CB             Borland C++Builder Demo
DEMO.PAS            Borland Delphi Demo
DoPE                WIN32 Libraries and DLL's
INCLUDE             Header Files

To run the DoPE demos doubleclick the EXE file in the appropriate directory.
To build the demos keep the directory structure. Header and library files 
will be included with relative paths from the INCLUDE and DoPE directory.
